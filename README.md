# ZevarByUs

Official website for ZevarByUs, a calm editorial jewellery experience built around the brand promise:

**Crafted For Every Soul**

## Tech

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide icons

## Run Locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
